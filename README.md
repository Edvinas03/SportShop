A full-stack e-commerce web application for sports products, built with React (frontend) and ASP.NET Core (backend).

Features

Product catalog with filtering by category, gender, and price

Product details with image gallery and size selection

Admin dashboard for adding, editing, and deleting products

User authentication with JWT

MySQL database integration

Deployment on Azure with CI/CD (GitHub Actions)

Responsive design using Tailwind CSS




Technologies Used:
